## Features

- Use **local images**
- Simple **timer-based training**
- Works in browser
- Adaptive UI (desktop / mobile)
- Language: russian. (english will be add soon)
- Single-file script

---


## Notes

- Works locally only  
- No external image sources  
- Beta version (0.2), may contain bugs
